The CitableLibrary

  • a library has a list of 0 or more collection objects implementing the CitableLibraryTrait
  • content can be selected from collections using URN logic
  • the library can be serialized to plain text in CEX format

This example builds a library with a single citable collection. The collection is of a type called ReadingList, and its contents are a list of ISBN values. (The following section on the CitableLibraryTrait shows you how to implement your own citable collection by walking through the implementation of the ReadingList type.)

citelib
Automatically assembled citable library <"urn:uuid:7939c302-5823-11ec-1f13-43a2532b2fa8">
Citable library with 1 collection

Serialize

A library can be serialized to CEX format.

cex(citelib) |> print
#!cexversion
3.0.2

#!citelibrary
name|Automatically assembled citable library
urn|"urn:uuid:7939c302-5823-11ec-1f13-43a2532b2fa8"
license|Creative Commons Attribution, Non-Commercial 4.0 License <https://creativecommons.org/licenses/by-nc/4.0/>

#!fakecexblock
urn:isbn:022661283X
urn:isbn:3030234134
urn:isbn:022656875X

Individual collections can be instantiated from complete CEX blocks.

Warning

fromcex is not yet implemented for an entire library. Follow this issue in the issue tracker.

Instantiate a ReadingList from a CEX block:

block = """#!fakecexblock
urn:isbn:022661283X
urn:isbn:3030234134
urn:isbn:022656875X
"""
fromcex(block, ReadingList)
Main.ReadingList(Main.IsbnUrn[urn:isbn:022661283X, urn:isbn:3030234134, urn:isbn:022656875X, ])
